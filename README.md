# Kvittordning

En native Android-start for en app som ska hjalpa till att samla och ordna kvitton.

## Kom igang

Bygg debug-appen:

```powershell
.\gradlew.bat assembleDebug
```

Installera pa en ansluten Android-enhet eller emulator:

```powershell
.\gradlew.bat installDebug
```

APK:n hamnar i:

```text
app/build/outputs/apk/debug/app-debug.apk
```
